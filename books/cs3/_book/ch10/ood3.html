Object Oriented Vocabulary

An object is something that contains both __ and __. It is an _ of a class.

A class is a __ for creating an object.

An instance is what you call a specific version of a __.

An _ method is a method called on an object that helps define the behavior of the class.

An _ variable is a variable beloging to an object that helps define the state of the class.

An _ method is a method called on the class. It does not have an object as the receiver. It is accessible to all the instances of the class.

An __ variable is a variable belong to the class, not any other instance. It is accessible to all instances of the class.

In Java, related classess are grouped into __

Objects are usually initialized by __.

__ is a reference to the current object.

__ varibles are accessible from outside of the class but it is not a good practice. We should use __ instead.
