u5.2 design student management system

5.2 Design Student Management System

1. Design Model:

1.1 Student has 3 columns

  • name: CharField, 30, unique
  • school: CharField, 100
  • email: CharField, 200, unique

2. Design pages

2.1 url: /students

  • breadcrumb: Students
  • actions: add student
  • list of students
    • name and link to edit
    • school
    • email
    • action link to delete

2.2 url: /students/new

  • breadcrumb: Students / new student
  • StudentForm

2.3 url: /students//edit

  • breadcrumb: Students /
  • StudentForm

2.4 url: /students//delete

  • breadcrumb: Students /
  • confirmation

3. s5/urls.py

    path('', views.handle_students, name="home"),

    path('students/', views.handle_students, name="students"),
    path('students/new/', views.handle_student_new, name="student_new"),
    path('students/<student_pk>/edit/',
         views.handle_student_edit,
         name="student_edit"),
    path('students/<student_pk>/delete/',
         views.handle_student_delete,
         name="student_delete"),

4. s5/views.py

# Create your views here.
def handle_students(request: HttpRequest):
    students = []
    return render(request, 's5/students.html',
                  context={'students': students})


def handle_student_new(request: HttpRequest):
    return render(request, 's5/student_new.html',
                  context={})


def handle_student_edit(request: HttpRequest, student_pk):
    student = {"pk": student_pk, "name": "Lucas"}
    return render(request, 's5/student_edit.html',
                  context={'student': student})


def handle_student_delete(request: HttpRequest, student_pk):
    student = {"pk": student_pk, "name": "Lucas"}
    return render(request, 's5/student_delete.html',
                  context={'student': student})
