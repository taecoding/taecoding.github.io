s06.6 Aggregation

Count

```
import random
random.seed(0)
a = [random.randint(1,6) for i in range(20)]
```

  • How many "1"s are in the list?

      count = 0
  for li in a:
      if li == 1:
          count += 1
  print(count)

  • How many 2 or 4 are in the list?

      count = 0
  for li in a:
      if li == 2 or li == 4:
          count += 1
  print(count)

  • How many 2,4,6 are in the List?

     print(3 in [1,2,3])

 count = 0
 for li in a:
     if li in [1, 2, 3]:
         count += 1
 print(count)

Total

  • scores
      import random
  random.seed(0)
  scores = [random.randint(50,100) for i in range(20)]
  print(scores)
  # total
  total = 0
  for li in scores:
      total += li
  print(total)

max and min

```python
maxV = scores[0]
for li in scores:
    if li > maxV:
        maxV = li
print(maxV)
```

indexOf

```python
a = [10, 30, 40, 15]

## indexOf
maxV = a[0]
maxI = 0

for i, li in enumerate(a):
    if (li > maxV):
        maxV = li
        maxI = i

print(maxI)
```
