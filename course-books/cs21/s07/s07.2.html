Exercise File

  1. Please write a friends management system that supports the following commands.

    • add: ask the given name and phone
    • del: delete the friend for the given name
    • check: print phone number for the given name
    • ls: list of all friends
    • done: stop the system

  2. Please write a fruit inventory system that supports the following commands.

    • sell: remove the quanity of the given fruit
      • if no fruit, not enough fruit
    • purchase: stock the quantity of the given fruit
    • check: print the quantity of the given fruit
    • ls: list of all fruits
    • done: exit the system

  3. Write a program that reads a file, breaks each line into words, strips whitespace and punctuation from the words, and converts them to lowercase.

     hint> 
 - whitespace, punctuation
 - replace and translate

  4. Then modify the program to count the total number of words in the book, and the number of times each word is used.

  5. Print the number of different words used in the book. Compare different books by different authors, written in different eras. Which author uses the most extensive vocabulary?

  6. Modify the program from the previous exercise to print the 20 most frequently-used words in the book.

  7. Modify the previous program to read a word list and then print all the words in the book that are not in the word list. How many of them are typos? How many of them are common words that should be in the word list, and how many of them are really obscure?

import string

def process_file(filename):
    h = dict()
    fp = open(filename)
    for line in fp:
        process_line(line, h)
    return h

def process_line(line, h):
    line = line.replace('-', ' ')

    for word in line.split():
        word = word.strip(string.punctuation + string.whitespace)
        word = word.lower()

        h[word] = h.get(word, 0) + 1

hist = process_file('emma.txt')


def total_words(h):
    return sum(h.values())

def different_words(h):
    return len(h)

print 'Total number of words:', total_words(hist)
print 'Number of different words:', different_words(hist)

def most_common(h):
    t = []
    for key, value in h.items():
        t.append((value, key))

    t.sort(reverse=True)
    return t

t = most_common(hist)
print 'The most common words are:'
for freq, word in t[0:10]:
    print word, '\t', freq


def subtract(d1, d2):
    res = dict()
    for key in d1:
        if key not in d2:
            res[key] = None
    return res
